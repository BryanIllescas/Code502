Documentación - Compilador de Música

Introducción

Bienvenido a la documentación del compilador de música. Aquí encontrarás información sobre cómo utilizar esta herramienta.

Instalación

Para utilizar el compilador de música, simplemente accede a nuestro sitio web desde cualquier navegador web compatible.

Uso

Puedes cargar código musical desde un archivo o escribirlo manualmente en el área de código. Luego, puedes realizar análisis léxicos y sintácticos para verificar la validez del código.
